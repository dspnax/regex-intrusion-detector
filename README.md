## Regex-Based Intrusion Detection System

### Abstract
A lightweight intrusion detection system that uses regular
expressions to detect common attack patterns in system logs.

### Features
- SQL Injection detection
- XSS detection
- Brute-force login detection

### Technologies
- Python
- Regular expressions
- Log analysis

### Limitations
Regex-based detection is fast but limited in expressive power.

### Future Improvements
- DFA optimization
- Machine learning integration
